PRei (P0)
storage

S3-like Storage for the Browser

IndexedDB powered. Local-first. Encrypted.

Features

folder Bucket & Object system
lock AES-GCM Encryption
offline_bolt Offline-ready storage
api S3-like API design

Quick Example


import { createStorage } from "./prei.js";

const p0 = await createStorage();

await p0.createBucket("files");

await p0.putObject({
  bucket: "files",
  key: "hello.txt",
  body: "Hello World"
});

const file = await p0.getObject("files", "hello.txt");
console.log(await file.body.text());

API Overview


p0.createBucket(name)
p0.putObject({ bucket, key, body })
p0.getObject(bucket, key)
p0.listObjects(bucket)
p0.deleteObject(bucket, key)
p0.protectObject({ bucket, key, password })

Encryption

lock AES-GCM with PBKDF2 key derivation 

await p0.protectObject({
  bucket: "files",
  key: "secret.txt",
  password: "mypassword"
});

const file = await p0.getObject("files", "secret.txt", {
  password: "mypassword"
});

Delete

delete Remove objects or entire buckets 

await p0.deleteObject("files", "old.txt");

await p0.deleteBucket("files");

Update (Rename / Replace)

edit Update via copy + overwrite pattern 

// rename
await p0.copyObject({
  from: { bucket: "files", key: "a.txt" },
  to: { bucket: "files", key: "b.txt" }
});

await p0.deleteObject("files", "a.txt");

// replace content
await p0.putObject({
  bucket: "files",
  key: "b.txt",
  body: "new content"
});

Unique Example

auto_awesome Local encrypted vault inside the browser 

await p0.createBucket("vault");

await p0.putObject({
  bucket: "vault",
  key: "note.txt",
  body: "Top secret data"
});

await p0.protectObject({
  bucket: "vault",
  key: "note.txt",
  password: "123"
});

// later
const file = await p0.getObject("vault", "note.txt", {
